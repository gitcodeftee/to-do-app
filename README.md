# 🎮 Gamified To-Do App

A modern task management web app with gamification features to make productivity engaging.

## 🚀 Live Demo
https://gitcodeftee.github.io/to-do-app/

## ✨ Features
- ✅ Add, edit, and delete tasks
- 🎯 Task completion tracking
- 🎮 Gamification (points & levels)
- 🌗 Dark/Light mode
- 🔄 Drag and drop reordering
- 🔍 Filters (All / Active / Completed)
- 💾 Local storage support
- 🎨 Glassmorphism UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
